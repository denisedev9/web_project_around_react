Around the U.S

Around the U.S es una aplicación web desarrollada con React.

El proyecto fue refactorizado completamente para trabajar bajo una arquitectura basada en componentes, utilizando estado, props y renderizado dinámico para actualizar la interfaz de manera eficiente.

La aplicación permite:

• Visualizar tarjetas de imágenes dinámicamente
• Agregar nuevas tarjetas mediante formularios controlados
• Editar la información del perfil
• Actualizar la foto de perfil
• Eliminar tarjetas
• Abrir imágenes en modales interactivos

La implementación con React permite una estructura modular, reutilizable y escalable, optimizando la experiencia del usuario y el manejo del estado de la aplicación.

Tecnologías utilizadas:

• React
• JavaScript
• HTML5
• CSS3
• Manejo de estado y componentes funcionales
